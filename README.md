# 🖥️ PT13 - Operating System Monitoring

## 📌 Description

This project is a practical assignment focused on monitoring operating systems in **Windows** and **Ubuntu/Linux** environments.
The goal is to understand how to analyze system performance, detect issues, and use both graphical and command-line monitoring tools.

---

## 🎯 Objectives

* Understand system **performance and reliability**
* Use monitoring tools in Windows
* Analyze processes and resources in Linux
* Interpret CPU, memory, disk, and network metrics
* Generate performance reports and statistics

---

## 🪟 Windows Monitoring

### 🔹 Tools Used

* Reliability Monitor
* Task Manager
* Resource Monitor
* Performance Monitor (PerfMon)

### 📊 Features Covered

* System stability index analysis
* Detection of system errors and critical events
* Process monitoring (CPU, RAM, disk, network)
* PID-based process management
* Creation of:

  * Data Collector Sets
  * Performance alerts
* Remote system monitoring

---

## 🐧 Ubuntu/Linux Monitoring

### 🔹 Commands Used

* `uptime`
* `top`
* `ps`
* `mpstat`
* `vmstat`
* `free`
* `df`
* `iostat`

### 📊 Features Covered

* System load analysis (load average)
* Process management and zombie processes
* CPU monitoring and idle time analysis
* RAM and SWAP memory supervision
* Disk usage analysis
* Disk I/O performance monitoring
* Data collection for statistics and graphs

---

## 🧪 Lab Work

The project includes hands-on exercises such as:

* Monitoring real-time system performance
* Creating and configuring performance logs
* Generating reports
* Simulating system activity to analyze behavior

## 🛠️ Technologies & Environments
